# 雅丽洁防晒乳 AIGC短片 — 分镜Prompt（纯净版）

> 说明：AI生成全身人物时最容易出现肢体变形/"火柴人"问题。这版做了两处调整：
> 1. 涉及人物的镜头全部改成**手部/侧脸/局部特写**，不做全身动作——局部特写的成功率远高于全身，这是目前AI视频生成的通用规律
> 2. 每条人物类prompt里都加了 **"真实人体比例，解剖结构正确，不要肢体变形"** 这类约束词，进一步降低出错概率
> 竖屏9:16，共7个镜头，约30秒

## 补充：如果生成的人物还是变形/像"火柴人"

1. **优先做纯手部/局部特写镜头**（1、3、4、5），这几个成功率最高，人物类镜头能少则少
2. **镜头6如果生成的剪影还是变形，直接改成真人实拍**——这种"逆光走出画面"类的全身动作AI目前普遍不稳定，实拍比死磕AI更省时间
3. **生成时如果工具支持"负面提示词"（negative prompt）**，加上：肢体变形、多余肢体、比例失调、僵硬机械动作、扁平二维感
4. **镜头4、6这类局部人物镜头，也可以只生成静态高质量图片，再用剪映做简单的呼吸感/缩放动效**，比强行让AI生成动态人物更可控
